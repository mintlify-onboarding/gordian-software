Gordian offers three methods of integration for maximum flexibility and ease

* _The Gordian Widget_ \- The Gordian API and the the Gordian front end Gordian's API + Gordian's proven widget _(recommended)_
* _The Gordian API_ \- Only the Gordian API (for cases in which you may have your own front end)
* _The Gordian White Label_ \- Post-booking micro site where Gordian handles payment via Stripe
